Suit Tracker is a system that tracks various data about a business.

### Repository Structure

- `/server` - Backend Typescript: Contains APIs, operations and table
  definitions
- `/client` - Frontend React Typescript application

### Technologies

- Backend: Typescript, nix, deno
- Frontend: React, Typescript, Vite

### Coding Standards

- Always prefer simple code where possible
- Avoid use of comments unless there is logic that is not obvious from first
  glance
- Follow the DRY principle
- Keep documentation concise and consistent
- Tests must be ordered alphabetically
- Tests use describe and it, and must follow patterns layed out in existing test
  files
- Use descriptive variable names and avoid abbreviations (i.e. don't use 'x' as
  a lambda parameter)
- Write clean and maintainable code
